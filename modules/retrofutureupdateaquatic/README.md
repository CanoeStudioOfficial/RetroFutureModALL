# RetroFuture Update Aquatic

1.13 content module for the RetroFuture series on Minecraft 1.12.2.

Reference sources:
- `参考/Oceanic-Expanse-master`
- `参考/Future-MC-main`
- `参考/Fluidlogged-API-1.12.2-Latest`

Fluidlogged API is intentionally not copied into this module. Put a compatible Fluidlogged API jar in this module's `libs/` folder for compile/run integration, and ship it as a separate mod dependency.

Initial migration targets:
- waterlogging/fluidlogged-compatible aquatic blocks
- corals, coral fans, sea grass, kelp, sea pickles
- conduit, trident, turtle, drowned, dolphins, fish variants
- ocean ruins, shipwreck-style loot hooks, buried treasure-style loot hooks
